# ✈️ Flight Booking System

A Python-based Flight Booking System for searching, booking, and managing flights.

## 🚀 Features
- Search flights
- Book tickets
- View bookings
- Database integration

## 🛠️ Technologies
- Python
- HTML, CSS, JavaScript
- MySQL

## ▶️ How to Run
1. Run server.py
2. Open login.html in browser
