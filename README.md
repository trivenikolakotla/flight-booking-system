# ✈️ Flight Booking System

## 📌 Overview
This project is a Flight Booking System developed using Python and web technologies. 
It allows users to register, login, search for flights, and manage bookings in a simple and efficient way.

---

## 🚀 Key Features
- 🔐 User Authentication (Login & Signup)
- 🔍 Flight Search Functionality
- 🧾 Ticket Booking System
- 🗄️ Database Integration using SQL
- 🌐 Simple Web Interface using HTML

---

## 🛠️ Tech Stack
- **Frontend:** HTML  
- **Backend:** Python  
- **Database:** SQL  

---

## ⚙️ How It Works
1. User creates an account or logs in  
2. Searches for available flights  
3. Books tickets  
4. Data is stored and managed using database  

---

## ▶️ How to Run
1. Install Python (3.x)
2. Setup database using `setup.sql`
3. Run the backend:
4. Open `login.html` in browser
